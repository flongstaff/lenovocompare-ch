**Part A: Create PhysicalSizeComparison.tsx**

Create `components/compare/PhysicalSizeComparison.tsx` as a new "use client" component.

Props:

```typescript
interface PhysicalSizeComparisonProps {
  readonly models: readonly Laptop[];
}
```

Implementation per user decisions:

1. **Proportional CSS rectangles** — width x depth rendered as scaled boxes, height as a thin side-view bar
2. **Side-by-side layout** — one rectangle per model in a row, using COMPARE_COLORS for borders
3. **All four dimensions shown**: width, depth, height, weight (label below each rectangle)
4. **Collapsible** via `useState<boolean>(true)` — starts expanded, user can collapse

Scaling approach (from research):

```typescript
const MAX_WIDTH_PX = 200;
const modelsWithDims = models.filter((m) => m.dimensions);
if (modelsWithDims.length === 0) return null; // graceful fallback

const maxWidth = Math.max(...modelsWithDims.map((m) => m.dimensions!.widthMm));
const scale = maxWidth > 0 ? MAX_WIDTH_PX / maxWidth : 1;
```

Per-model rectangle:

- Width: `Math.round(m.dimensions.widthMm * scale)` px
- Height (visual): `Math.round(m.dimensions.depthMm * scale)` px (depth = front-to-back = visual height of top-down view)
- Use `border: 2px solid ${COMPARE_COLORS[index]}` for the rectangle
- Background: semi-transparent fill using COMPARE_COLORS with low opacity

Height side-view bar:

- A thin horizontal bar below the rectangle showing relative height (thickness)
- Scale: same proportional approach but with a separate MAX for height values
- Width of bar matches the rectangle width, height of bar proportional to heightMm

Labels below each model:

- Model name (truncated if long)
- Width x Depth x Height in mm
- Weight from `formatWeight(m.weight)`

If a model in the compare set has NO dimensions, show a "Dimensions unavailable" placeholder in that model's slot instead of an empty box.

Style using Carbon dark theme classes and Tailwind. Use `.carbon-card` for the collapsible container. Collapse toggle: ChevronDown/ChevronUp from lucide-react.

Section title: "Physical Comparison" with the collapse toggle.
